Feladat
-------

Egy snake programhoz mesterséges intelligencia készítése. 
Az elkészített AI által vezérelt snake-nek meg kell próbálni 
legyőzni a mintaként adott AI-t! A túlélés mellett fontos,
 hogy törekedjen az étel megszerzésére.
A forráskód legyen dokumentálva!


A csomag tartalma
-----------------

- Forráskód
- Referencia hivatkozásokhoz szükséges állományok
- A java verzió dokumentációja

Forráskód:
A forráskód Visual Studio 2010-zel készült. A benne található logika két mintai-t inicializál, a második helyettesítendő az elkészített AI-val.

Referencia hivatkozásokhoz szükséges állományok:
A Kukac projecbe, referencia hivatkozásként 2 dll van hivatkozva: lib\KukacCommon.dll, lib\MintaAI.dll. 

A java verzió dokumentációja:
Indítás az doc\index.html fájllal történik. A dokumentáció olvasását ajánlott az Interfaces/Adat interfésszel kezdeni. Fontosabb interfészek, enumok:
Enums/PalyaElemek, Interfaces/Test, Enums/Iranyok, Interfaces/Ai.
